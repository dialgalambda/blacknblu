# blacknblu
steam-inspired theme for aliucord

# REQUIRED PLUGINS FOR OPTIMAL LOOKS!

### [better status indicators by juby210](https://github.com/Juby210/Aliucord-plugins/raw/builds/BetterStatusIndicators.zip)
enable radial statuses, then adjust the status colours:

online: #55B9F3

idle: #3F6D95

do not disturb: #2EC735

optional: enable avatar status and chat status platforms

### [square avatars by juby210](https://github.com/Juby210/Aliucord-plugins/raw/builds/SquareAvatars.zip)

set round corners radius to 0

### [compact mode by zt64](https://github.com/zt64/aliucord-plugins/raw/builds/CompactMode.zip)

no extra settings required. 

# other things to know about

the theme will use GNU unifont, which is optional and you can change it to whatever font you want!
